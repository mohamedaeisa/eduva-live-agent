- [ ] Verify traceability: binary fingerprint from Head + Heart + Tail.
- [ ] Controlled sequential concurrency to prevent serverless timeouts.

## Phase 2: AI Atom Extraction
- [ ] Use v2.2 Master Prompt in `atomExtractionService.ts`.
- [ ] Strict JSON schema compliance (metadata, core, extended, quizPotential).

## Phase 5: Hybrid Grading Implementation

- [ ] Online: Semantic similarity via AI embedding comparison.
